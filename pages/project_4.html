Web : Détection de fraudes

Analyser les tickets remontés par les caisses

Objectif

Application web à disposition des employés du siège et des points de vente Picard pour faciliter l'analyse des tickets de caisse. L'application est utilisée pour détecter les fraudes, vérifier les retours clients ou pour tout autre cas de consultation des transactions magasins.

Fonctionalités

  • Charger une journée de tickets pour le point de vente sélectionné
  • Utiliser les filtres prédéfinis pour réduire le périmètre
  • Recherche des valeurs dans les tickets
  • Imprimer le résultat des recherches
  • Exporter le résultat des recherches
