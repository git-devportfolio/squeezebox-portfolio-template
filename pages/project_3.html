SAP FIORI : Le réassort

Application mobile à destination des 1200 points de vente Picard

Objectif

Tous les jours le manager doit réapprovisionner la surface de vente pour garder une disponibilité continuelle des produits et maximiser les ventes. Ce procédé appelé réassort est une priorité pour le manager dans le cadre du développement ou de la survie de son activité commerciale. A l'aide de la technologie FIORI, le manager utilise un iPad directement sur la surface de vente pour identifier et noter les produits à sortir du stock avant de se rendre en réserve pour sortir physiquement la marchandise. Après validation des saisies, le système recalcule automatiqement le stock de la réserve magasin.

Fonctionalités

  • Lister par défaut les produits disponibles en réserve
  • Filtrer les produits pour ajuster le périmètre article proposé
  • Scanner les codes produits pour accéder rapidement à un produit
  • Conserver les saisies automatiquement dans SAP pour limiter les pertes de données
  • Valider les saisies et mettre à jour le stock
  • Appliquer un balisage de couleur sur les produits
